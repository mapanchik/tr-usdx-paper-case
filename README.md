# tr-usdx-paper-case
Paper craft enclosure for the famous (tr)uSDX QRP transceiver by DL2MAN/PE1NNZ. Perfect as a temporary solution while you wait for a 3d-printed case, or just for a cool custom look

## Specifications and printing
**Scale:** 1:1 (Make sure to print at **100% scale!!**, DO NOT SELECT "Fit to page")
**Paper weight:** Use strictly **180 g/m² paper**, this is a critical! Standart 80g paper is too weak, and thicker 200g cardstock is too hard to fold cleanly!

## Assembly instructions
1. Carefully cut along the solid outer lines
2. Cut out the holes for the display, encoder, buttons and side connectors.
3. Score along the lines using a ruler and a blunt tool like an epmty pen to get clean, sharp folds
4. Fold and join the edges using a Super glue (Cyanoacrylate)

## ⚠️ CRITICAL GLUE WARNING!
Apply **STRICTLY 1 TINY** drop of super glue per side
Liquid cyanoacrylate flows extremly fast. If you use too much, it will leak inside, ruin the electronics, destroy the plastic connectors, and create ugly white fogging on the OLED display
Less is more - on **SMALL** drop is more than enought to bond 180g paper instantly.
